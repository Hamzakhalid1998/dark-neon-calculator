# Dark Neon Calculator

## Description

A modern, dark-themed calculator built with HTML, Tailwind CSS, and JavaScript.  
This calculator performs basic arithmetic operations with a sleek neon UI, perfect for web portfolios and learning DOM manipulation.

## Features

- Dark neon glow design
- Responsive layout for different screen sizes
- Basic arithmetic operations: `+`, `-`, `*`, `/`, `%`
- Clear button (`C`) to reset input
- Equals button (`=`) to calculate results
- Prevents multiple consecutive operators (e.g., `7++3`)
- Error handling for invalid expressions

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript

## How to Use

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Click buttons to input numbers and operations.
4. Press `=` to calculate the result.
5. Press `C` to clear the input.
